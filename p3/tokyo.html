Tokyo Website

RIT - Web and Mobile II

This was a project that we built for our midterm and then added additional PHP code to for our final project in my Web and Mobile II course. We were asked to make a website about our favorite city or town that we've been to and then talk about it. I chose Tokyo, my all-time favorite city! We needed to add a comments page, a feedback page, and several other things while storing information visitors left in a database.
