Thanks for Visiting my Project 3 Site!

The font, Karla, that I used on this site was designed by Jonny Pinhorn. The Javascript used to make the dropdown menu on the home page go directly to the page was adjusted from a sample Jim Habermas taught in my Web and Mobile II Class. The coding I personalized for the hamburger menu was developed by Terence Budu-Biney and edited/customized by me

Overall, I'm much happier with the design of this site than I was with Project 2. I used a lot of CSS Grid again, but I did NOT use grid-template-rows like I made the mistake of doing last time. I also used grid areas rather than indivual positioning. I thought that it made it a lot easier to control and position object within a media query, but I was disappointed once I dug into it that I wasn't able to get the same conent overlay effect that I'd used in project 2 (in particular on the homepage). I still feel that the site is so much more responsive and looks SO much better as it's resized than it did in my previous version (which also had like 6 media queries but still looked not-so-good!)

I did not use any template or framework at all

I used Javascript in the same ways as I did in Project 2, but I redid the entire CSS code and HTML code from scratch for this iteration of the project in order to correct all the issues that left me feeling so frustrated in the last version. I used the same content, but did update a few things because of how I've progressed on projects since project 2.
